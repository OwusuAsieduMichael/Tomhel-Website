# School images

Drop your photos here. They are served from the site root, e.g.:

- `public/images/hero/campus.jpg` → `/images/hero/campus.jpg`

Suggested subfolders:

| Folder | Use for |
|--------|---------|
| `hero/` | Home page banner |
| `about/` | About page, leadership, history |
| `academics/` | Classrooms, programs |
| `facilities/` | ICT lab, library, playground |
| `student-life/` | Sports, clubs, events |
| `gallery/` | Gallery page (campus, sports, graduation, etc.) |
| `news/` | News & events thumbnails |
| `logo/` | School logo and brand assets |

After uploading, tell us which files go where and we can wire them into the site.
